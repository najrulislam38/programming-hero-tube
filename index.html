Q. Discuss the scope of var, let, and const

Ans. Var first came as a variable in JavaScript and was added to const 2015 (ES6)
Var: Var variable is a global scope variable. Variable declared 'var' are hosted to their containing function or global scope. Var variable can be re-declare and reassigned.
let: Let variable is a block scope variable. It cannot redeclare. but let variable can be reassigned.
const: The const variable is a block scope variable. Its cannot be re-declare or re-assigned. Const variable used whose variable thats value will not change.
